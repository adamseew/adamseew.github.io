Constrained Optimization

So far... (Unconstrained Optimization)

In a simple optimization problem without constraints we search for the \(m\) state values \(u_1,u_2,...u_m\) that minimize a performance index (called also objective function or performance index) \(\mathcal{L}\), which is a function of these parameters $$\mathcal{L}(u_1,u_2,...u_m).$$ The vector \(\mathbf{u}=\begin{pmatrix}u_1 & u_2 & \cdots & u_m\end{pmatrix}^T\) of the state values is also called decision vector. Without any constraints on states and with the performance index \(\mathcal{L}(\mathbf{u})\) that has both first and second order partial derivate everywhere, necessary conditions for a minimum given \(i=1,...,m\) are $$\frac{\partial\mathcal{L}}{\partial u_i}=0\label{1}\tag{1}$$ $$\frac{\partial^2\mathcal{L}}{\partial u_i^2}\geq 0\tag{2}.$$ The last condition means that the \((m\times m)\)-matrix, whose components are \(\partial^2\mathcal{L}/\partial u_i\partial u_j\) must be positive semidefinite, i.e., have eigenvalues that are zero or positive.

All points that satisfy the first condition are called stationary points. The sufficient condition for a local minimum is the first necessary condition for a minimum (Equation \(\ref{1}\)), and, the eigenvalues are strict positive (that means \(\partial^2\mathcal{L}/\partial u_i^2>0\)).

If \(\partial\mathcal{L}/\partial u_i=0\) is satisfied, but \(\partial^2\mathcal{L}/\partial u_i^2=0\), that is the determinant of the matrix is zero (or in other words that one or more of its eigenvalues is zero), we need further analysis know when or not the point is a minimum. Such a point is called singular point. Let's take as an example a linear function \(\mathcal{L}(\mathbf{u})\) that we want to minimize. A linear function has \(\partial^2\mathcal{L}/\partial u_i^2=0\) in every point and thus does not have a minimum.

Equality Constraints

A more general class of problems involves the minimization of \(m\) decision parameters of a performance index which is function of \((n+m)\) parameters $$\mathcal{L}(\mathbf{x};\mathbf{u}),$$ where \(n\) state parameters \(\mathbf{x}=(x_1,x_2,...,x_n)\) are determined by decision parameters through set of \(n\) constraint relations $$f_1(\mathbf{x};\mathbf{u})=0,...,f_n(\mathbf{x};\mathbf{u})=0.$$
