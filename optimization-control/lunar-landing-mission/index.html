Sequential Quadratic Programming (SQP)

Lunar Landing Mission



Optimization problems are used to obtain an exact solution (within certain tolerance) in a number of important scientific applications, such as optimal control of system's driven by complex differential dynamics and path planning of (un)manned aerial vehicles. So far we studied techniques to implement the optimization theory to a wide range of mathematical problems that we classified in two different categories.

Problems with linear cost function and constraints belong to the category of linear problems and can be solved involving Linear Programming or LP. There are two different approaches that we studied to solve LP problems, the Simplex Method and the Interior-Point Method.

Problems with non linear quadratic cost function and linear constraints are part of the class of quadratic problems and can be solved using Quadratic Programming or QP. We have studied the Sequential Quadratic Programming Method that solves a QP problem at each iterate until it reaches a (tolerable) solution.
